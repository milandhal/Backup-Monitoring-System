Scheduling Guidelines
  • Off-Peak Hours: Schedule during 02:00–05:00 AM to avoid load on active databases.
  • MySQL Only: Live mysqldump works only for MySQL instances. Oracle instances require a custom dump agent.
  • Cron Jobs: Schedules run as server-side Node.js cron daemons — no external scheduler needed.
  • Auto-Logging: Every scheduled run is automatically logged in Backup History with size, duration, and MD5 hash.
  • Pause/Resume: Use the schedule table below to temporarily suspend a task without deleting it.
